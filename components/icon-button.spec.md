# Icon Button — 仕様
アイコンのみの操作ボタン。
- **実装**: `.fig-icon-button`（トークン経由）
- **States**: Default/Hover/Active/Disabled/Focus
- **a11y**: `aria-label` 必須、フォーカス可視、最小タップ 44px
- **Profiles**: 3プロファイル成立
- **Tokens**: `--fig-target-min` / `--fig-radius-control` / `--color-text-secondary` / `--color-surface-container-low`
- **Preview**: `preview/components-buttons.html`（要整備）
