# FAB（Floating Action Button）— 仕様
浮遊する主要アクション。
- **実装**: `.fig-fab`（トークン経由）
- **States**: Default/Hover/Active/Disabled/Focus
- **a11y**: `aria-label` 必須（アイコンのみ）、フォーカス可視、最小タップ 44px 以上
- **Profiles**: 3プロファイル成立
- **Tokens**: `--color-surface-brand` / `--color-text-onBrand` / `--fig-radius-pill` / `--fig-elevation-floating` / `--fig-target-comfortable`
- **Preview**: `preview/components-buttons.html`（要整備）
