# Side Sheet — 仕様
側面からのシート（主に Web-Admin の詳細/フィルタ）。
- **実装**: `.fig-side-sheet`（トークン経由）
- **States**: 閉/開

- **Profiles**: 3プロファイル成立（Admin 主用途）
- **Tokens**: `--color-surface-default` / `--fig-elevation-floating` / `--elevation-z-modal`
- **Preview**: `preview/surface-layering.html`（要整備）
